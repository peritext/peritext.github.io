Fonio

Latest version: March 2020
Fonio is a collaborative editor of static websites for students in Human and Social Sciences, created by the médialab Sciences Po and funded by the FORCCAST program.
Fonio is designed to promote good documentation practices among students, by allowing to describe all the resources gathered during an inquiry in a rich and homogeneous way (date, source, authors,...). It also allows the creation of an extensive glossary to describe key terms and consult them interactively.
Fonio is collaborative and allows several authors to work simultaneously on contents, design and documentation. The software is organized according to a decentralized logic in which access to the tool is not assured via a single platform, but rather consists of the deployment of small instances (called "classrooms"), each corresponding to a specific course and existing for a limited period of time. All the students' works are saved and archived privately after the end of the semester using the Bocal tool. However, the application also allows students to download their site at any time to save it or to publish it online in a clean and simple way.

Métis

Latest version: January 2018
Métis was an experiment aimed at producing an editorial platform for an Arts scholarly journal, carried out in collaboration with the Hybrid Publishing research group at the École Nationale Supérieure des Arts Décoratifs. It was an opportunity to genericize the Peritext model and to develop some of its current concepts, as well as to experiment the mobilization of all the Peritext technical modules in the framework of a centralized application using a database.
